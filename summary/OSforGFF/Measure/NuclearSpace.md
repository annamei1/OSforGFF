**Informal**: A continuous linear map $T : E \to F$ is **nuclear** if there exist sequences
$(\varphi_n) \subset E^*$ and $(y_n) \subset F$ such that
$\sum_n \lVert \varphi_n \rVert \cdot \lVert y_n \rVert < \infty$ and
$T(x) = \sum_n \varphi_n(x)\, y_n$ for all $x$.

## Hilbertian Seminorms

def Seminorm.IsHilbertian {E : Type*} [AddCommGroup E] [Module ℝ E]
    (p : Seminorm ℝ E) : Prop
```

**Informal**: A seminorm $p$ on a real vector space $E$ is **Hilbertian** (i.e., comes from
an inner product) if it satisfies the parallelogram law:
$$p(x+y)^2 + p(x-y)^2 = 2\bigl(p(x)^2 + p(y)^2\bigr) \quad \forall\, x, y \in E.$$

    (p q : Seminorm ℝ E) : Prop
```

**Informal**: The canonical inclusion $\hat{E}_p \hookrightarrow \hat{E}_q$ (from the Hilbert
completion with respect to $p$ into that with respect to $q$, where $q \leq p$) is
**Hilbert-Schmidt** if the sums $\sum_{i} q(e_i)^2$ remain uniformly bounded over all
finite $p$-orthonormal sequences $(e_i)$. For diagonal operators with eigenvalues
$(\sigma_k)$, this reduces to $\sum_k \sigma_k^2 < \infty$.

## Nuclear Spaces

### [`NuclearSpace`](../../OSforGFF/Measure/NuclearSpace.lean#L126) — Definition

**Lean signature**
```lean
class NuclearSpace (E : Type*) [AddCommGroup E] [Module ℝ E] [TopologicalSpace E] : Prop
```

**Informal**: $E$ is a **nuclear space** if its topology is generated by a countable family
of Hilbertian seminorms $(p_k)_{k \in \mathbb{N}}$ such that for every $k$ there exists
$m > k$ with the inclusion $\hat{E}_{p_m} \hookrightarrow \hat{E}_{p_k}$ being
Hilbert-Schmidt. The composition of two consecutive Hilbert-Schmidt inclusions is nuclear
(trace-class), giving the Grothendieck-Pietsch characterization for Fréchet spaces.

## Schwartz Space is Nuclear

### [`schwartz_nuclear`](../../OSforGFF/Measure/NuclearSpace.lean#L145) — Definition *(axiom)*

**Lean signature**
```lean
axiom schwartz_nuclear {E F : Type*}
    [NormedAddCommGroup E] [NormedSpace ℝ E] [FiniteDimensional ℝ E]
    [NormedAddCommGroup F] [NormedSpace ℝ F] :
    NuclearSpace (SchwartzMap E F)
```

**Informal**: The Schwartz space $\mathscr{S}(E, F)$ is a nuclear space. The justification
(accepted here as an axiom) is that the Sobolev-Hermite norms
$\lVert f \rVert_k^2 = \sum_n (1+n)^{2k}\lvert\langle f, h_n\rangle\rvert^2$
are Hilbertian, generate the Schwartz topology, and the inclusion from level $k+1$ to
level $k$ has eigenvalues $(1+n)^{-1}$ satisfying $\sum_n (1+n)^{-2} = \pi^2/6 < \infty$,
hence is Hilbert-Schmidt (Gel'fand-Vilenkin Vol. 4, Thm 3; Trèves Ch. 51).
