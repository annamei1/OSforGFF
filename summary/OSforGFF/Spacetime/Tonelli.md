## Overview

Proves a Tonelli factorization theorem for Schwartz functions on spacetime $\mathbb{R}^4$:
for Schwartz functions $f, g$ and a bounded nonnegative measurable kernel $K(t_1, t_2)$
depending only on the time coordinates, the double spacetime integral factors as
$$\int\!\int \lVert f(x)\rVert\,\lVert g(y)\rVert\,K(x_0,y_0)\,dx\,dy = \int\!\int K(t_1,t_2)\,G_f(t_1)\,G_g(t_2)\,dt_1\,dt_2,$$
where $G_f(t) = \int_{\mathbb{R}^3}\lVert f(t,v)\rVert\,dv$ is the spatial norm integral.
The proof uses the spacetime decomposition from `Decomposition.lean`, Fubini–Tonelli,
and an integral swap. This is a key tool for the OS3 reflection positivity bounds.

### [`schwartz_tonelli_spacetime`](../../OSforGFF/Spacetime/Tonelli.lean#L112) — Theorem

**Statement**: For Schwartz functions $f, g$ on $\mathbb{R}^4$ and a bounded nonnegative measurable kernel $K : \mathbb{R} \times \mathbb{R} \to \mathbb{R}$ depending only on time coordinates:
$$\int_{x}\!\int_{y} \lVert f(x)\rVert\,\lVert g(y)\rVert\,K(x_0, y_0)\,dx\,dy = \int_{t_1}\!\int_{t_2} K(t_1, t_2)\,G_f(t_1)\,G_g(t_2)\,dt_1\,dt_2,$$
where $G_f(t) = \int_{\mathbb{R}^3}\lVert f(t,v)\rVert\,dv$ and $G_g(t) = \int_{\mathbb{R}^3}\lVert g(t,v)\rVert\,dv$.
